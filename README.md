	Réseau Social des Alumni
 Description
Ce projet est une application web de réseau social dédiée aux alumni développée dans le cadre d’un projet universitaire à l’Université de Djibouti.
L’application permet aux utilisateurs (étudiants et anciens) de consulter des profils, publier du contenu et interagir en ligne, tandis que les administrateurs peuvent gérer les utilisateurs, les publications et les interactions via une interface dédiée.
Objectifs
Automatiser la gestion des interactions entre alumni
Éviter les contenus dupliqués ou incohérents
Centraliser les données (utilisateurs, publications, interactions)
Offrir une interface simple et intuitive
